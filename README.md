# Software2OA

This project was built with the limitation of using only Java API's and frameworks in JAVA 8. The UI, built with JavaFX, allows the adding, updating, and deleting of appointments within a database connected to SQL. 

Other functions of the program include:
 - Adding, editing and deleting customers
 - Generating various reports of appointment types, customer records, etc.
 - Real-time search
 - User login
 - Various levels of exception controls
